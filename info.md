# Material Thermostat Card

A Material 3 Expressive thermostat card for Home Assistant, designed to pair with
`material-you-theme` and inspired by the Google Home / Nest thermostat UI.

- Draggable circular temperature dial, current temperature, and step buttons
- Climate HVAC / fan / swing selectors as an icon row or dropdown
- Per-option label & icon customization from the visual editor
- Material 3 tokens with graceful Home Assistant theme fallbacks

See the [README](https://github.com/lageorgem/ha-material-thermostat-card) for configuration and the
feature roadmap.
